# Password Generator Dashboard

An interactive password generator built with **Streamlit**. Generate strong, secure passwords using three different modes: Pin, Random, and Memorable.

## Features

- **Pin Password:** Generate numeric passwords of custom length.  
- **Random Password:** Generate passwords with letters, numbers, and symbols.  
- **Memorable Password:** Generate easy-to-remember passwords using words, with customizable separators and capitalization.  

## How to Run

1. Clone the repository:
    ```bash
    git clone https://github.com/shaqayeqNz/streamlit-dashboard
    cd your-repo
    ```

2. Install dependencies:
    ```bash
    pip install streamlit
    ```

3. Run the Streamlit app:
    ```bash
    streamlit run dashboard.py
    ```

4. Open the URL provided by Streamlit in your browser to start generating passwords.

## Usage

- Choose a password type from the radio buttons.
- Customize options using sliders, toggles, and text inputs.
- Your password will be displayed immediately.

## Requirements

- Python 3.7+  
- Streamlit
